# Todoアプリ

やることを入力すると、リストとして表示されるシンプルなTodoアプリを作成しました。

## 機能
- タスクの追加
- タスクの削除
- データの保存（localStorage）
- お問い合わせフォーム

## 使用技術
- HTML
- CSS（Flexbox）
- JavaScript

## 工夫した点

データ保存機能の実装に力を入れました。  
localStorageを用いた保存処理について、setItemやgetItemの仕組みを調べながら実装しました。

また、配列データをそのまま保存するのではなく、JSON.stringifyとJSON.parseを用いることで、データを正しく保存・取得できるように工夫しました。

さらに、お問い合わせフォームではrequired属性を使用し、空欄のまま送信できないようにすることで、入力ミスを防ぐようにしました。

実装の過程では、JavaScriptの読み込み位置や処理の順序によって動作が変わる点に苦戦しましたが、原因を調べて修正し、正しく動作するように改善しました。
